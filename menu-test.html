🚀 How to Add New Pages

Step 1: Create your new page (e.g., pricing.html)

Step 2: Open auto-menu-generator.js

Step 3: Add to the menu structure:

{ id: 'pricing', file: 'pricing.html', label: 'Pricing' },

Step 4: Save and deploy! The menu updates automatically.
